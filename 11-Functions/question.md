# 11 - Functions Important Questions

## 1. What is a function?

### Answer

Function is a reusable block of code that performs a specific task.

---

## 2. Difference between parameter and argument?

### Answer

Parameter:

```js
function add(a,b)
```

`a` and `b` are parameters.

Argument:

```js
add(3,5)
```

`3` and `5` are arguments.

---

## 3. What does return do?

### Answer

Stops function execution and returns a value.

---

## 4. What is default parameter?

### Answer

Default value used when no argument is passed.

Example:

```js
function greet(name = "Guest")
```

---

## 5. What is rest operator?

### Answer

Collects multiple values into an array.

```js
function sum(...nums)
```

---

## 6. Can functions accept objects?

### Answer

Yes.

Objects can be passed as arguments.

---

## 7. Can functions accept arrays?

### Answer

Yes.

Arrays can be passed as arguments.
