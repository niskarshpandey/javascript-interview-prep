# Variables

## const
- Cannot be reassigned
- Block scoped
- Must be initialized

Example:
const id = 1

## let
- Can be reassigned
- Block scoped

Example:
let name = "Niskarsh"
name = "Pandey"

## var
- Can be reassigned
- Function scoped
- Avoid using in modern JS

Example:
var age = 22

## Undefined

let state
console.log(state) // undefined
